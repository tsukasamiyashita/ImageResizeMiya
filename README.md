# ImageQualityMiya v1.2.0

画像の品質（圧縮率）を一括で変更するためのシンプルなデスクトップアプリです。
JPEGやWebPなどのファイルサイズを削減したい場合に便利です。

## 更新情報 (v1.2.0)
- **バージョン情報表示**: メニューバーからアプリのバージョンと更新履歴を確認できるようになりました。
- **UI調整**: タイトルバーにバージョン番号を表示するようにしました。

## 主な機能
- **複数ファイル対応**: ドラッグ＆ドロップやファイル選択で複数の画像を一括処理できます。
- **画質選択**: 「最高画質」から「限界画質(Quality 1)」まで幅広く選択可能。デフォルトは「低画質(50)」です。
- **EXIF自動補正**: スマホ等で撮影した写真の向き（回転）を自動で修正して保存します。
- **保存設定**: 元のフォルダに保存するか、別のフォルダを指定するか選べます。
