# ImageResizeMiya

画像の品質（圧縮率）やファイル形式を一括で変更するためのデスクトップアプリケーションです。
JPEGやWebP、PNGに加えて、iPhone等で撮影された **HEIF/HEIC形式** の読み書きにも対応しており、ファイルサイズを効果的に削減できます。

## 主な機能
* **複数ファイルの一括処理**: ドラッグ＆ドロップで複数画像を一気に読み込めます。
* **ファイル形式の変換機能**: 元の形式のまま保存するだけでなく、JPEG、PNG、WebP、HEIF形式への一括変換が可能です。
* **HEIFフォーマット対応**: `.heic` や `.heif` 形式の読み込みおよび品質を落とした再保存が可能です。
* **EXIF自動補正**: スマホ等で撮影した写真の向き（回転情報）を自動で補正して保存します。
* **柔軟な保存先**: 元のフォルダに接尾辞（`_q50` など）を付けて保存するか、別のフォルダにまとめて保存するか選べます。「元の画質」を選択した場合は接尾辞が `_org` になります。
